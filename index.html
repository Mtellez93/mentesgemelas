Mentes Gemelas

Prepara la pantalla principal, comparte el lobby con un QR y deja que cada jugador entre desde su celular con su nombre y código. Cuando todos estén conectados, el host puede iniciar la partida.

Start Game

Abre la vista del host para mostrar el código del lobby, esperar a todos los jugadores y pulsar Start Game cuando estén listos.

🎮 Abrir host

Join Lobby

Los jugadores escanean el QR o abren la pantalla de unión, escriben su nombre y el código del lobby, y entran directo a la partida.

📱 Unirse al juego
1. El host comparte el lobby en la TV o monitor.
2. Cada jugador entra con su nombre y el código mostrado.
3. Cuando haya un número par de jugadores, el host puede comenzar.
